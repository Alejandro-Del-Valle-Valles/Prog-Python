# Práctica 04: Torneo de los Tres Magos
---
> Por Alejandro del Valle Vallés

La práctica consiste en practicar la creación y manejo de clases en **Python** mediante un torneo de Magos del universo de Harry Potter, donde se pueden crear Magos,
Campeones, y Dragones para que se enfrenten.
Tanto el código como la documentación del mismo están escritos completamente en español.

## Elementos practicados:
- Clases
  - Clases Abstractas y No Abstractas
  - Clases como servicio
  - Clases para POO
- Herencia
  - Herncia múltiple y simple.
- Importación de clases desde módulos
